# Galería Visual — Dragon Ball

**Descripción breve:**  
Sitio estático con una galería temática de Dragon Ball (20+ recursos), usando una paleta inspirada en trajes, auras y personajes, con gradientes, overlays y buenas prácticas de legibilidad.

## Ver en local
- **Requisitos:** Navegador actualizado.
- **Pasos:**
  1. Clona: `git clone "https://github.com/DeivyAcosta1523-pixel/galeria-visual-Deivy-Acosta.git"
  2. Entra: `cd galeria-visual-Deivy`
  3. Abre `index.html` en tu navegador.

## GitHub Pages
- **URL pública:** https://deivyacosta1523-pixel.github.io/galeria-visual-Deivy-Acosta/

## Decisiones de diseño
- **Paleta de colores:** Naranja (traje de Goku), azul (Capsule Corp/Blue), verde (Shenlong) y dorado (SSJ) para transmitir energía y contraste.
- **Gradientes:** Header con degradado cálido (naranja/dorado) para identidad; banner radial (azul/verde) para sensación de aura y foco visual.
- **Optimización de imágenes:** Comprimir JPG/PNG (~70–80% calidad), usar `loading="lazy"`, nombres descriptivos y tamaños ajustados al grid.

## Estructura
- `assets/img/`: imágenes
- `assets/css/styles.css`: estilos
- `index.html`: página principal
